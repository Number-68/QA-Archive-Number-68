# TC - 01: Install Theme From File
# Prerequisites:
- A successfully built Blender application from the official source repository.
    
- A working internet connection.
    
- A web browser installed.
    
- Access to **extensions.blender.org**.
    
- Ability to extract `.zip` files on the system.
    
- Blender binary accessible (e.g., via terminal).

# Test Steps:
- **Open the browser** and navigate to **extensions.blender.org**.
    
- In the Extensions site, **navigate to the Themes section**.
    
- Select **any theme** from the list.
    
- Scroll to the section containing the **Download** link (the link located under “Drag and drop into Blender”).
    
- Click the **Download** link to obtain the theme `.zip` file.
    
- Open the system’s file explorer and **locate the downloaded** `.zip` **file**.
    
- **Extract** the `.zip` file (“Extract Here” or equivalent).
    
- **Launch Blender** from the terminal using the compiled Blender binary.
    
- When Blender opens, the **splash screen** should appear.
    
- Click **General** to open a new default project.
    
- In the top-left application menu bar, click **Edit**.
    
- From the dropdown, select **Preferences**.
    
- In the Preferences window, select **Themes** from the left sidebar.
    
- In the Themes panel, click the **Install…** button (top-right).
    
- In the file browser dialog, navigate to the folder containing the extracted theme file.
    
- Select the `.xml` **theme file** and click **Install Theme**.

# Expected Results:
- Blender should immediately **apply the newly installed theme** upon installation.
    
- The UI color scheme should change to match the downloaded theme.
    
- The installed theme should now appear in the **Themes** list within Preferences.
    
- No errors or warnings should appear during installation.

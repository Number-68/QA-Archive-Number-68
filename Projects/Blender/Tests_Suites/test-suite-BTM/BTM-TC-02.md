# TC - 02: Install Theme From "Get Extensions"
# Prerequisites:
- A successfully built Blender application from the official source repository.
    
- A working internet connection.
    
- Blender binary accessible (e.g., via terminal).
    
- Access to the **Get Extensions** feature inside Blender Preferences.

# Test Steps:
- **Launch Blender** from the terminal using the compiled Blender binary.
    
- When the splash screen appears, click **General** to open a new default project.
    
- In the top-left application menu bar, click **Edit**.
    
- From the dropdown menu, select **Preferences**.
    
- In the Preferences window, select **Get Extensions** from the left sidebar.
    
- At the top of the Get Extensions panel, open the **Filter by Type** dropdown and select **Themes**.
    
- In the **Repository** dropdown, ensure that **extensions.blender.org** is selected as the source.
    
- In the search bar, type the name of any theme you want to install.
    
- In the search results, locate the theme card and click the **Install** button.
    
- After installation completes, navigate to the **Themes** tab in the left sidebar of Preferences.
    
- Open the theme selection dropdown and verify that the newly installed theme appears in the list.
    
- Select the installed theme and confirm that Blender’s UI updates to match the theme’s design.

# Expected Results:
- The selected theme should install successfully through the Get Extensions interface.
    
- Blender’s UI should immediately update to reflect the newly installed theme.
    
- The installed theme should appear in the **Themes** dropdown list within Preferences.
    
- No errors, warnings, or failed installation messages should occur.
